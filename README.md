# game-wiki-skills

Claude AI skills built from game wiki pages, designed to help Claude answer in-game questions accurately — from mechanics and item stats to combat and crafting.

## Skills

### 🧀 Milky Way Idle (`skills/milky-way-idle`)

Covers all aspects of [Milky Way Idle](https://milkywayidle.com) (available on [Steam](https://store.steampowered.com/app/2081600/Milky_Way_Idle/)):

- Gathering: Milking, Foraging, Woodcutting
- Production: Cheesesmithing, Crafting, Tailoring, Cooking, Brewing, Alchemy
- Enhancing: success rates, protection mechanics, the Observatory
- Combat: stats, styles, abilities, XP distribution
- Monsters: drop tables, zones, bosses, combat tiers
- Houses: all 17 rooms with upgrade costs and buffs

## Usage

These skills are intended for use with Claude's skill system. Drop the relevant skill folder into your skills directory and Claude will use it to answer game-related questions with accurate, reference-backed data.
