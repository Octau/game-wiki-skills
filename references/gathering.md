# Gathering Skills: Milking, Foraging, Woodcutting

Gathering skills produce raw materials. All are affected by global Gather Quantity
and Experience buffs from Houses. Each has its own gear and a dedicated House room.

Common gathering teas/effects:
- Skill Tea / Super / Ultra: Efficiency bonus + temporary level boost (300s).
- Gathering Tea: +15% gather quantity (300s).
- Wisdom Tea: +12% experience (300s).
- Processing Tea: 15% chance to convert gathered resource into processed material (300s).
- Efficiency Tea: 10% chance to repeat the action instantly (300s).

General skilling gear (applies broadly): Necklace/Philosopher's of Wisdom (+exp),
Necklace of Speed (+skilling speed), Rings/Earrings of Gathering (+gather qty),
Essence Find and Rare Find jewelry, Collector's Boots (+10% Efficiency at Lv.60
Milking/Foraging/Woodcutting).

## Milking

Main way to gain types of Milk, used in Cheesesmithing, Cooking, and Brewing.
Equip Brushes to increase Milking speed.

Cows by level: Verdant Cow (Lv.1, 5 XP), Cow (Lv.1 base), Azure Cow (Lv.10, 7.5 XP),
Burble Cow (Lv.20, 12.5 XP), Crimson Cow (Lv.35, 20 XP), (Lv.50, 30 XP),
Rainbow (Lv.65, 40 XP), Holy Cow (Lv.80, 55 XP).

Teas: Milking Tea (+2% eff, +3 lvl), Super Milking Tea (+4% eff, +6 lvl),
Ultra Milking Tea (+6% eff, +8 lvl).

Dairy Barn (House room): each level +1.5% Milking Efficiency, +0.05% Wisdom,
+0.2% Rare Find, +1.5% instant-repeat chance, +0.05% exp, +0.2% drop rate of
meteorite caches/artisan's crates/treasure chests.

Milking-specific gear: Dairyhand's Top (+10% Eff, +15% Rare Find, Lv.90),
Dairyhand's Bottoms (+10% Eff, +4% Exp, Lv.90).

Foraging

Master the skill of picking up things.

Main Drop

**Foraging** is a gathering skill in Milky Way Idle. The player can gain Experience to level up this skill by foraging in various locations. There are seven foraging locations that the player can unlock by leveling up Foraging. The time it takes to forage increases accordingly to the tier of the locations. The player can reduce the time it takes to forage by using Shears. Various pieces of Equipment can boost Foraging efficiency, experience gain, and gathering quantity.

Foraged resources have multiple uses: Crafting, Tailoring, Cooking, and Brewing.

## Formulas

### Action Time

The time to forage a location is calculated as:

| Action Time Formula |
| --- |
| `Action Time = Base Time / (1 + Foraging Speed)` |

**Foraging Speed Sources:**

| Source | Bonus |
| --- | --- |
| Shears | +15% to +105% (base values, scales with enhancement) |
| Necklace Of Speed | +4% base (9.8% at +10) |
| Philosopher's Necklace | +4% base (9.8% at +10) |

**Example:**

| Component | Value |
| --- | --- |
| Base Time (Farmland) | 6 seconds |
| Azure Shears Speed | +30% |
| **Final Action Time** | **6 / (1 + 0.30) = 4.62 seconds** |

### Experience

Wisdom and equipment experience bonuses increase the experience you receive from skilling actions. All Wisdom sources are additive with each other, then multiply the base experience.

| Skilling Experience Formula |
| --- |
| `Final XP = Base XP × (1 + Wisdom + Equipment Experience + Charm Experience)` |

**Foraging-Specific Equipment:**  Forager's Bottoms (Lv 90, +4% base (5.16% at +10)),  Celestial Shears (Lv 90, +4% base (5.16% at +10)), Foraging Charm (scales with tier and enhancement)

See the Wisdom page for all sources and detailed examples.

### Gathering Quantity

Gathering increases the number of resources you receive from each location. Gathering bonuses are multiplicative with the base yield range.

| Gathering Formula |
| --- |
| `Quantity = Random(min-max) × (1 + Gathering Bonus)` |

See the Gathering page for all sources and how this stat works.

### Efficiency

Efficiency allows you to instantly repeat actions without consuming time. At higher Efficiency percentages, you get guaranteed repeats plus a chance for additional actions.

**Skill-Specific Equipment:**

| Forager's Bottoms | 90 | +10% base (12.9% at +10) |
| Collector's Boots | 60 | +10% base (12.9% at +10) |

See the Efficiency page for the full formula, all sources, and examples.

### Processing

Processing Tea provides a chance to automatically convert gathered resources into processed materials:

| Processing Formula |
| --- |
| `Processing Chance = 15% (from Processing Tea)` |

When processing triggers, resources are instantly converted to their processed forms (e.g., Cotton to Thread, Wheat to Flour), saving time and providing processing skill experience.
